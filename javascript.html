JavaScript (ES2015+)

AirBnB JS Standards FTW

We've adopted AirBnB's JavaScript style guide wholesale, except where it directly contradicts our existing ES5 standards. In those cases, our standards supercede.

File naming conventions

A few guidelines on how to name your files for consistency:

  • If the file contains a class or module that is exported, it should be named the same as the module. So the file exporting the class Account should be called Account.js.
  • If the file has no classes/modules or is a collection of utilities, it should use a camel-case name starting with a lower-case letter, e.g. numberFormatUtils.js or chartSettingsDefaults.js.
  • All folder names should be lower-spine case, e.g. data/models/user-preferences/UserPreference.js.
